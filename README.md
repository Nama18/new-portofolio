# Portfolio

My personal portfolio website built with Next.js.

## Live Website

**[new-portofolio-two.vercel.app](https://new-portofolio-two.vercel.app)**

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- GSAP (animations)
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
